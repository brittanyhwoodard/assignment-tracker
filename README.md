# Assignment Tracker
A straightforward easy to use assignment tracker made for students. 

#Features
- Full CRUD functionality
- Persistent local storage
- Modal-based add/edit interface
- Dynamic rendering
- Responsive layout

#Tech Stack
- HTML
- CSS
- Vanilla JavaScript

#Future Improvements
- Custom color coding for each assignment
- List sorting options
- Calendar view
- Import/export
- User theme customization
